# Portfólio de Análise de Dados e Business Intelligence

Este repositório apresenta projetos de Análise de Dados e Business Intelligence
desenvolvidos com foco em tomada de decisão e visão de negócio.

Os projetos demonstram habilidades em coleta, tratamento, análise e visualização
de dados, utilizando ferramentas amplamente usadas pelo mercado.

## Ferramentas utilizadas
- Python (Pandas, Matplotlib)
- SQL
- Power BI
- Excel

## Projetos
- Dashboard de Faturamento, Custos e Lucro (Power BI)
- Análise Exploratória de Dados de Serviços (Python)
- Consultas SQL para Análise de Dados

## Objetivo
Demonstrar capacidade de transformar dados em informações úteis para apoiar
decisões estratégicas em empresas de diversos setores.
